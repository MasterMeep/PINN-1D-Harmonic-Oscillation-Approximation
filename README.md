# PINN-1D-Harmonic-Oscillation-Prediction
Using a PINN the network learns to predict the position of a 1d harmonic oscillator using 10 ground truth data points (sampled from the ground truth) and 100 collocation points for the pde loss function

# Example -- 0.0004 loss -- 99.99% accuracy -- 20,000 epochs -- lr 1e-3
